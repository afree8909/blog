View测量流程

  1. measure
    • 性质：final方法，View测量初始方法
    • 作用：基本逻辑计算，是否重新测量，调用onMeasure，缓存测量规格
  2. onMeasure
    • 性质：测量方法，子类可复写实现自己的测量方式
    • 作用：进行测量规格计算，调用getDefaultSize和setMeasureDimension
  3. getDefaultSize
    • 作用：根据View的测量规格计算宽/高值
  4. setMeasureDimension
    • 作用：存储测量后的宽/高值
